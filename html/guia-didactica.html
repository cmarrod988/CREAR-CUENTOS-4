CREAMOS CUENTOS DIGITALES

Proyecto de creación de cuentos digitales para 2º de Primaria

GUÍA DIDÁCTICA

Guía didáctica del proyecto

Esta guía didáctica recoge la relación entre las actividades del proyecto, los criterios de evaluación y las competencias trabajadas durante la creación de cuentos digitales en 2º de Educación Primaria.

Además, incluye orientaciones metodológicas y recomendaciones para facilitar la aplicación del proyecto en el aula.

METODOLOGÍA

El proyecto se desarrolla mediante metodologías activas y participativas, favoreciendo el aprendizaje cooperativo, la creatividad y el uso responsable de las tecnologías digitales.

Las actividades están adaptadas al alumnado de 2º de Primaria y siguen principios del Diseño Universal para el Aprendizaje (DUA).

RECOMENDACIONES PARA EL PROFESORADO

Uso guiado de herramientas digitales.
Apoyo visual constante.
Actividades cooperativas.
Adaptación de tareas según el ritmo del alumnado.
Uso del panel interactivo y recursos manipulativos.
